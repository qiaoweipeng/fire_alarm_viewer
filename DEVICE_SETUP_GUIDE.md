# 火灾报警设备查看器 - 设备配置指南

## 概述

这个应用已经为您实现了三大核心功能：
1. **显隐切换** - 通过顶部按钮控制所有设备名称的显示/隐藏
2. **悬停高亮** - 在隐藏状态下，鼠标放在设备上会高亮显示该设备和名称
3. **搜索定位** - 在搜索框输入设备名称，会自动高亮显示匹配的设备

现在您需要根据实际的火灾报警设备图片，手动添加设备的坐标和名称。

---

## 如何添加设备

### 第一步：打开设备配置文件

打开文件：`client/src/lib/devices.ts`

您会看到一个 `devices` 数组，其中包含示例设备。

### 第二步：理解坐标系统

```
图片左上角 (0, 0)
        ↓
        +────────────────────→ X 轴（向右递增）
        │
        │
        │
        ↓ Y 轴（向下递增）
```

- **X 坐标**：从图片左边开始计算，向右递增（单位：像素）
- **Y 坐标**：从图片顶部开始计算，向下递增（单位：像素）
- **width**：设备高亮框的宽度（像素）
- **height**：设备高亮框的高度（像素）

### 第三步：测量设备坐标

**最简单的方法 - 使用浏览器开发者工具：**

1. 打开应用，在浏览器中按 `F12` 打开开发者工具
2. 点击左上角的"检查元素"按钮（或按 `Ctrl+Shift+C`）
3. 将鼠标放在图片中的设备上
4. 在开发者工具中，您会看到该元素的坐标信息
5. 记录下 X、Y 坐标和设备的宽度、高度

**或者手动测量：**

1. 使用图片编辑软件（如 Photoshop、GIMP）打开图片
2. 使用测量工具或标尺工具，记录设备的位置和大小
3. 记录下像素坐标

### 第四步：添加设备到配置文件

打开 `client/src/lib/devices.ts`，在 `devices` 数组中添加新设备：

```typescript
export const devices: Device[] = [
  {
    id: "device_1",           // 唯一标识符，建议使用 device_序号 格式
    name: "火灾报警主机",     // 设备的中文名称
    x: 50,                    // 设备在图片中的 X 坐标（像素）
    y: 100,                   // 设备在图片中的 Y 坐标（像素）
    width: 150,               // 高亮框的宽度（像素）
    height: 120,              // 高亮框的高度（像素）
  },
  // 继续添加其他设备...
];
```

### 第五步：验证配置

1. 保存文件后，应用会自动刷新
2. 在应用中查看设备是否正确显示
3. 尝试：
   - 点击"隐藏名称"按钮，然后将鼠标放在设备上，应该能看到高亮效果
   - 在搜索框中输入设备名称，应该能看到对应设备被高亮

## 常见问题

### Q: 设备高亮框不在正确的位置怎么办？

**A:** 调整 `x`, `y`, `width`, `height` 的值。建议：
- 先设置一个较大的 `width` 和 `height`（如 200x200）
- 在应用中查看高亮框的位置
- 根据需要微调 `x`, `y` 的值
- 最后调整 `width` 和 `height` 以完美匹配设备大小

### Q: 搜索功能不工作怎么办？

**A:** 检查以下几点：
- 确保设备的 `name` 字段填写正确
- 搜索时输入的名称必须与 `name` 字段中的文本匹配（支持模糊搜索）
- 检查是否有拼写错误或多余的空格

### Q: 如何删除一个设备？

**A:** 在 `client/src/lib/devices.ts` 中，找到对应的设备对象，删除整个对象即可。

### Q: 如何修改设备的名称？

**A:** 在 `client/src/lib/devices.ts` 中，找到对应设备的 `name` 字段，修改文本即可。

## 编辑模式 - 快速录入坐标

为了让您更快速地添加设备，应用提供了一个**编辑模式**，可以直接在图片上框选设备，自动生成坐标数据。

### 使用编辑模式

1. **启用编辑模式**
   - 点击顶部的"编辑模式"按钮（橙色按钮）
   - 图片边框会变成橙色，鼠标光标会变成十字形

2. **框选设备**
   - 在图片上按住鼠标左键并拖动，框选一个设备
   - 您会看到一个虚线框显示您正在框选的区域
   - 框的右上角会实时显示宽度和高度

3. **获取坐标**
   - 松开鼠标后，会弹出一个对话框显示坐标信息
   - 同时坐标数据会打印到浏览器控制台
   - 按 `F12` 打开开发者工具，在控制台中查看完整的设备配置代码

4. **复制配置**
   - 在控制台中找到生成的设备配置代码
   - 复制 JSON 格式或 TypeScript 格式的代码
   - 粘贴到 `client/src/lib/devices.ts` 的 `devices` 数组中
   - 修改 `name` 字段为实际的设备名称

5. **退出编辑模式**
   - 点击"退出编辑"按钮返回正常模式

### 编辑模式的控制台输出示例

```
=== 新设备配置 ===
复制以下代码到 devices.ts 的 devices 数组中：

{
  "id": "device_1716097200000",
  "name": "新设备",
  "x": 150,
  "y": 200,
  "width": 120,
  "height": 100
}

TypeScript 格式：
{
  id: "device_1716097200000",
  name: "设备名称", // 请修改为实际的设备名称
  x: 150,
  y: 200,
  width: 120,
  height: 100,
},
==================
```

### 编辑模式的优势

✅ **快速定位** - 不需要手动计算坐标，直接框选即可  
✅ **实时反馈** - 框选时实时显示宽度和高度  
✅ **自动生成** - 自动生成标准的设备配置代码  
✅ **批量录入** - 可以快速连续添加多个设备  

### 编辑模式的工作流程

```
1. 点击"编辑模式"按钮
   ↓
2. 在图片上框选一个设备
   ↓
3. 松开鼠标，查看坐标信息
   ↓
4. 打开控制台（F12），复制生成的代码
   ↓
5. 粘贴到 devices.ts，修改设备名称
   ↓
6. 重复步骤 2-5，添加更多设备
   ↓
7. 点击"退出编辑"按钮，返回正常模式
```

### 常见问题

**Q: 为什么框选后没有看到坐标信息？**

A: 请检查以下几点：
- 确保您在编辑模式下（按钮应该是"退出编辑"）
- 确保您拖动了鼠标（不是只点击了一下）
- 打开浏览器控制台（按 F12），查看是否有输出信息

**Q: 坐标数据不准确怎么办？**

A: 编辑模式生成的坐标是基于鼠标位置的，可能会有几个像素的误差。您可以：
- 在控制台中手动调整坐标值
- 或者在应用中查看框选框的位置，然后微调坐标值

**Q: 如何修改已添加的设备的坐标？**

A: 直接编辑 `client/src/lib/devices.ts` 文件中对应设备的 `x`, `y`, `width`, `height` 值即可。
